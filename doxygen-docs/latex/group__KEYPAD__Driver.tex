Defines the GPIO port and pins for the keypad. 

The following macros are used to configure the keypad\+:
\begin{DoxyItemize}
\item KEYPAD\+\_\+\+PORT\+: GPIO port used for the keypad.
\item KEYPAD\+\_\+\+ROWS\+: Number of rows in the keypad.
\item ROW0, ROW1, ROW2, ROW3\+: GPIO pins for each row.
\item KEYPAD\+\_\+\+COLS\+: Number of columns in the keypad.
\item COL0, COL1, COL2\+: GPIO pins for each column. 
\end{DoxyItemize}
