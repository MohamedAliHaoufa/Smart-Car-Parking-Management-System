{\em uint8} & Returns {\ttfamily GPIO\+\_\+\+RETURN\+\_\+\+LOCK\+\_\+\+OK} if the pin configuration is successfully locked, or {\ttfamily GPIO\+\_\+\+RETURN\+\_\+\+LOCK\+\_\+\+ERROR} if the locking process fails.\\
\hline
\end{DoxyRetVals}
\begin{DoxyNote}{Note}
The locking sequence is as follows\+:
\begin{DoxyItemize}
\item Write 1 to the lock key bit.
\item Write 0 to the lock key bit.
\item Write 1 to the lock key bit.
\item Read the lock key bit (should be 0).
\item Optionally, read the lock key bit again (should be 1 to confirm the lock).
\end{DoxyItemize}
\end{DoxyNote}
